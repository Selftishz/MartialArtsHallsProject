<h1 align="center">MartialArtsHalls project</h1>
<p>This project is a RESTful service that simulates the administration of the database of martial arts halls</p>
<h2>Technology stack:</h2>
<ul>
  <li>Java 17</li>
  <li>Maven</li>
  <li>Spring Core</li>
  <li>Spring Web</li>
  <li>Hibernate</li>
  <li>PostgreSQL 16</li>
</ul>
